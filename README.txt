# StagWeb
Esse sistema foi desenvolvido para o processo de seleção da empresa Sólides onde foi solicitado a criação de um WebApp para estagiários que apontam horários de trabalho.

Obs: aberto a sugestões e melhorias.

Funcionalidades
Acompanhamento de horários
Registro de horários

Pré-requisitos
Para testar o sistema ou o computador deve conter os seguintes softwares:

PHP 5.3 ou superior
Servidor Web 
My SQL 
Bootstrap e jQuery
